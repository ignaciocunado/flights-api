# Flights API

## Running the app

First, run the backend via

```bash
./gradlew bootRun
```

Then, navigate to the `/client` directory and run the frontent development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
